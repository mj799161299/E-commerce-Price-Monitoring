## 快速开始

### 1. 启动服务

```powershell
# 启动所有容器
docker-compose up -d

# 初始化数据库和测试
.\init.ps1
```

### 2. 访问应用

- 前端：http://localhost
- 后端API：http://localhost:8000
- API文档：http://localhost:8000/docs

### 3. 测试流程

1. 注册账号
2. 登录获取Token
3. 配置平台API凭证
4. 添加商品监控
5. 查看监控列表

详细测试步骤请参考 `API_TEST.md`

## 技术亮点

1. **前后端分离架构**：清晰的职责划分，便于维护和扩展
2. **异步编程**：FastAPI + SQLAlchemy异步操作，性能优异
3. **数据加密**：敏感信息AES加密存储，安全可靠
4. **JWT认证**：无状态认证，适合分布式部署
5. **Docker容器化**：一键部署，环境一致性
6. **类型安全**：TypeScript + Pydantic，减少运行时错误
7. **适配器模式**：统一的平台API接口，易于扩展新平台
