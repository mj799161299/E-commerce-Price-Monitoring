## 📝 使用指南

### 快速开始

```powershell
# 1. 启动后端服务
docker-compose up -d backend db

# 2. 启动前端开发服务器
cd frontend
npm install
npm run dev

# 3. 访问系统
# 前端: http://localhost:3000
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```

### 基本流程

1. **注册账号** → 填写用户名、邮箱、密码
2. **登录系统** → 进入仪表盘
3. **配置API** → 配置电商平台API凭证
4. **添加商品** → 输入商品链接，设置阈值
5. **配置推送** → 设置降价提醒渠道
6. **开始监控** → 系统自动监控价格变化

## ⚠️ 注意事项

### 1. 安全配置
生产环境部署前务必修改：
- 数据库密码（docker-compose.yml）
- JWT密钥（backend/.env）
- AES密钥（backend/.env，必须32字节）

### 2. API凭证申请
需要自行申请：
- 淘宝联盟: https://open.taobao.com/
- 京东联盟: https://union.jd.com/
- 多多进宝: https://jinbao.pinduoduo.com/

### 3. 推送配置
- Server酱: https://sct.ftqq.com/
- 企业微信/钉钉/飞书: 创建群机器人获取webhook

### 4. 开发模式

- 生产部署使用 Docker 构建
